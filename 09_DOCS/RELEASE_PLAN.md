# Release Plan

## Goal
Turn one source visual + one song + one lyric sheet into a release-ready media pack.

## Phase 1 — Inputs
- Confirm song final master is approved
- Confirm source clip is stable and loop-friendly
- Confirm lyrics are final
- Confirm artist/title metadata is correct

## Phase 2 — System prep
- Validate ffmpeg / ffprobe / font paths
- Normalize filenames
- Generate or refresh loop variants
- Generate styled subtitles
- Generate title cards
- Analyze audio envelope + beatmap
- Build timeline + render manifest

## Phase 3 — Master render
- Render section clips
- Concat section clips into one video master
- Combine full song audio
- Burn lyrics onto final master
- Generate soft subtitle version

## Phase 4 — Distribution exports
- 16:9 main release
- 9:16 vertical
- 1:1 square
- 15s / 30s / climax teasers

## Phase 5 — Release QA
- Check duration match
- Check audio exists in all intended exports
- Check subtitle burn quality
- Check safe areas for vertical crop
- Check file sizes and playback
- Check title / metadata / naming consistency

## Phase 6 — Release bundle
- Package final deliverables
- Write delivery manifest
- Write release readiness report
- Archive config and metadata with outputs

## Recommended additions before public release
- custom font choice aligned to branding
- polished title cards
- manual lyric timing refinement for key emotional lines
- custom teaser picks based on strongest hook moments
- thumbnail / cover art render
- YouTube description and credits file
